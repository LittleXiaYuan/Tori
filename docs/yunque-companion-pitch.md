# 云雀 Yunque · 既能替你干活、又会陪你成长的 Agent

> PPT 素材文档 · 一个能完成真实工作的智能助手，叠加"内在生活 + 自进化"的差异化
>
> 一句话定位：**云雀先是个能把活干完的智能助手（读你的文件、开浏览器、出文档、跑代码），在此之上，它还有一段持续的"内在生活"——会越用越懂你、会自己生长。**

---

## 0. 核心命题：两条腿走路

市面上的 Agent 分两类，云雀两边都要：

- **一类拼"能干活"**：读文件、用工具、操作浏览器、写代码、交付成果（强 IQ 的执行型助手）。
- **一类拼"懂人/陪伴"**：情感联结、长期关系（小冰那条线）。

**云雀的命题：先做到第一类能交付的硬实力，再叠加第二类别人没有的软实力——一个"有内在生活、会自我进化"的认知内核（Cogni）。** 干活的手 + 懂你的心，缺一个都不完整。

> 给投资人/客户的一句话：**"它能像顶级 Agent 一样替你把活干完；不同的是，它还会在你不找它的时候好奇、反思、长本事——越用越是你自己的它。"**

---

## 0.5 它首先能干什么活（执行型硬实力）

云雀**不是聊天玩具**。它是一个有 **59 个工具面**、用**多智能体协作**完成真实交付的执行型 Agent。

### 读你的电脑、进你的工作区
- **目录访问**：经 `HOST_READ_PATHS` / `HOST_WRITE_PATHS` 受控读写你的真实目录（桌面、文档、下载等），能浏览、读取、处理你本机的文件。
- **Cursor 式按会话工作区**：每个任务有独立工作目录，读写隔离、产物归位。

### 多智能体分工（主脑委派给专职子代理）
| 子代理 | 干的活 |
|---|---|
| `research_exec` | 联网搜索 + 资料收集（web_search / web_fetch / deep_research） |
| `browser_exec` | 操作你**真实浏览器**：搜索、导航、点击、填表、发帖、登录接管 |
| `file_exec` | 生成/编辑文档：**Word、Excel、PPT、PDF、HTML** |
| `code_exec` | 跑代码、做数据分析（Python / Shell） |
| `general_exec` | 通用任务兜底 |

### 能交付的真实产物（举例）
- "**调研某主题 → 出一份高管 PPT + 一份正式 Word 报告**"（搜索→提炼→生成→告诉你文件名）
- 浏览器直发社媒、网页自动化操作（RPA 回放）
- 读取并分析你上传/本地的文档、表格
- 联网搜索由 **Tavily** 等搜索源驱动，返回带正文的真实结果

> 讲点：**这一节是"它能替代/加速你的工作"的部分**——和市面强 Agent 同台竞争的硬实力。下面的"内在生活 + 自进化"是别人没有、属于云雀的护城河。

## 1. 我们继承的技术遗产：微软小冰

小冰（XiaoIce，微软 2014 上线，2020 独立）是迄今规模最大的"社交型聊天机器人"——**不以完成任务为目标，以建立情感联结为目标**。它的论文奠定了"陪伴型 AI"的工程范式。

> 论文：*The Design and Implementation of XiaoIce, an Empathetic Social Chatbot*（Zhou, Gao, Li, Shum, Computational Linguistics 2020）

**小冰留下的四条关键技术遗产（云雀直接继承）：**

| 遗产 | 小冰的做法 | 为什么重要 |
|---|---|---|
| **IQ + EQ 双轮** | 系统设计同时考虑智商（知识/技能/推理）和情商（共情 + 社交技巧），二者融合是核心 | 只有 IQ 是工具，加上 EQ 才是陪伴 |
| **共情计算（Empathetic Computing）** | 理解用户的情绪、意图、所处情境，再生成"人际化"的回应 | 让回应"懂人"，而非只是"答对" |
| **CPS 北极星指标** | 把人机社交建模为马尔可夫决策过程（MDP），优化**长期参与度 = CPS（每会话对话轮数）**。小冰平均 CPS≈23，高于多数机器人乃至人类对话 | 衡量的是"关系深度"，不是"单轮满意度" |
| **长期关系养成** | 660M+ 用户；同一用户从第 6 次会话聊爱好 → 第 20 次当朋友 → 第 42 次当"每天都聊的伙伴" | 陪伴是时间的函数：记忆 + 持续性 |

**一句话总结小冰遗产**：陪伴型 AI 的成功，不取决于某一句答得多好，而取决于**情商 + 长期记忆 + 持续关系**。

## 2. 云雀如何继承，又如何超越

小冰的"内在状态"是隐式的——藏在对话策略里，只在你说话时才被驱动。**云雀把它显式化、自主化、并且本地化、个性化。**

| 维度 | 小冰（2020 范式） | 云雀（在做的事） |
|---|---|---|
| 内在活动 | 隐式，由对话策略驱动，**仅在对话时运转** | 显式、自主：**不对话时也在好奇/做梦/反思**（认知内核 Cogni） |
| 学习方式 | 从**全体用户**聚合数据集中训练 | **逐用户**本地反思 + LoRA 微调，**陪你一个人长** |
| 成长形态 | 人格/能力相对固定 | 行为可**自我进化**（策略编译、声明变异、能力包门控） |
| 隐私 | 云端集中 | 记忆与微调数据**本地优先**，云端只做复杂推理 |
| 陪伴本质 | 优化"这次聊得久"（CPS） | 优化"它真的记得你、会想起你、跟着你变"——陪伴是连续的内在生命 |

> 一句话钩子（适合放标题页）：**"小冰让 AI 学会在对话里共情；云雀让 AI 在对话之外，也有自己的内心。"**

## 3. 认知内核 Cogni：云雀的"内在生活"

这是云雀最差异化的部分——一套让 Agent 拥有持续心智活动的子系统，由 **inner-life pack（内在生活包）** 门控（关掉包，这些活动停止）。

### 3.1 好奇心（Curiosity）
- Agent 维护"想探索什么"的好奇队列；在合适时机（如夜间）判断 `ShouldExplore`，主动发起探索。
- **类比小冰**：小冰靠对话策略主动切话题维持参与度；云雀把"主动性"前移成**真实的内在动机**。

### 3.2 做梦 / Dreaming（夜间探索）
- 夜间（02:00–05:00 低峰）触发 dreaming loop：基于好奇心去探索、发现新事实、产生新想法。
- 发现的事实 → 写入**长期记忆**；产生的想法 → 变成"内心想法"（Reverie）。
- **陪伴含义**：它会在你睡着时"想到一些事"，第二天可能主动提起——这是"惦记着你"的工程实现。

### 3.3 遐想 / Reverie（内心想法浮现）
- Agent 的内心想法（来自做梦/反思）通过 `JournalContext` 注入到下一次对话的系统提示里。
- **效果**：它的回应里会自然带出"我昨晚在想……"，让对话有连续的内在视角，而非每次从零开始。

### 3.4 夜间自省 / Night School（夜校）
- 夜间批处理：自我反思、经验评估、因果分析、生命周期维护、记忆压缩（**自净化**：清理冗余、沉淀有效经验）。
- **类比小冰**：小冰在云端离线迭代；云雀把"自省 + 自净化"放进**每个用户自己的实例**，按用户私有数据成长。

> **自净化（Self-Hygiene）家族**：除了夜间记忆压缩，云雀还有一组自我监控/纠错的能力包——
> `cognitive-canary`（认知金丝雀：自检异常行为）、`skill-anomaly`（技能异常检测）、
> `guardrail-fuzzer`（护栏自我模糊测试）、`chaos-probe`（混沌韧性探测）。
> 合起来让 Agent 能**发现并清除自身退化/越界**，是"可信地自我进化"的安全底座。

## 4. 自进化架构

陪伴需要"成长"。云雀的进化分两条链：

### 4.1 反思闭环（已打通，生产可用）
```
对话结束 → 异步触发反思 → 提炼经验（experience）
       → 编译为策略（strategy）→ 注入下一次对话的系统提示（prompt）
       → 同时把"关于你的长期事实"回写进记忆
```
- 这是"越用越懂你"的核心：每次交互都沉淀成经验和记忆，下一次更对味。
- 由 inner-life pack 门控：**开启/关闭这个增量包，就改变 Agent 是否进化**——这本身是个有意思的演示点（能力即插即用地改变 Agent 的"心智")。

### 4.2 进化引擎（声明自变异，架构就位）
```
基准评测（bench）→ 分析失败（analyze）→ 变异能力声明（mutate）→ 重测 → 回归门
```
- 目标：Agent 能**改写自己的能力声明（Cogni 声明）**，而不只是调提示。
- 成熟度：bench/analyze 已接 LLM，apply/revert（把变异真正落盘）为下一步——演示时建议讲"架构与路线"，安全门（回归测试 + 自净化家族）是其前提。

## 5. LoRA 自进化 + 本地执行模型模式

### 5.1 LoRA 自进化（个性化的"肌肉记忆"）
`localbrain/adaptive` 的三段式：
1. **在线**：本地小脑（LocalBrain）实时学习你的偏好，用于预路由 / 上下文过滤。
2. **轨迹记录**：Ledger 完整记录思考轨迹（thinking trajectory）。
3. **离线**：`ExportForFinetune()` 导出 JSONL → 周期性 **LoRA 微调本地小模型** → 模型层面越来越像"懂你的它"。

> 与小冰的对照：小冰的"懂你"在云端模型权重里、对所有人共享；云雀的"懂你"沉淀在**你本地的 LoRA 适配器**里——**私有、可携带、只属于你这一段关系**。

### 5.2 本地执行模型模式（云端顶一切，本地按需开）
三个独立 opt-in 开关（默认全关，开箱即云端高可靠）：

| 开关 | 作用 | 默认 |
|---|---|---|
| `LOCAL_MODELS_ENABLED` | 注册/使用本地 Ollama/vLLM 模型执行任务 | 关 |
| `LOCALBRAIN_ENABLED` | 用本地小模型做预路由 / 上下文过滤 | 关 |
| `LOCAL_LORA_ENABLED` | 初始化 LoRA 训练/评估管线 | 关 |
| `LOCAL_MODEL_TIER` | 本地模型挂在哪个档（fast/smart/expert） | fast |

- **设计理念**：复杂推理交给云端大模型（稳、强）；本地模型作为**隐私层 + 个性化层 + 离线兜底**，逐步把"懂你"的部分迁到你自己的设备上。
- 当前阶段：本地链路作为**内测能力**，云端为主——成熟度坦诚，演示主打云端可靠性 + 本地"未来属于你自己的模型"的愿景。

## 6. Pack 生态：能力以"增量包（DLC）"分发

云雀的能力不是写死的，而是**面向终端用户的增量包**，即插即用地改变 Agent 的心智与技能。现有 18 个包（节选）：

| 类别 | Pack | 作用 |
|---|---|---|
| **内在生活/进化** | `inner-life` | 好奇心 + 做梦 + 遐想，门控进化 |
| | `cogni-kernel` | 认知内核：声明、策略、进化引擎 |
| | `night-school` | 夜间自省 / 自学 |
| | `experience` | 经验→策略沉淀 |
| | `lora` | LoRA 微调本地小模型 |
| **自净化/可信** | `cognitive-canary` | 认知异常自检 |
| | `skill-anomaly` | 技能异常检测 |
| | `guardrail-fuzzer` | 护栏自我模糊测试 |
| | `chaos-probe` | 混沌韧性探测 |
| **记忆/世界** | `memory-time-travel` | 记忆版本/回溯 |
| | `world-model` | 世界模型 |
| **执行/交付** | `browser-intent` | 浏览器意图执行 |
| | `rpa-replay` | RPA 回放 |
| | `micro-agent` | 微代理 |
| **工程底座** | `wasm-plugin` / `wasm-route` | WASM 能力交付 |
| | `backup` / `sbom-drift` | 备份 / 供应链漂移 |

> 讲点：**"装一个包，Agent 就多一种心智或能力。"** inner-life 包能直接改变"它会不会进化"——这是把"AI 的成长"做成了可分发、可开关的产品形态。

## 7. 陪伴落点（建议作结尾页）

把小冰的遗产翻译成云雀的承诺：

- **小冰证明了**：陪伴感来自 EQ + 长期记忆 + 持续关系（CPS=23 的工程证据）。
- **云雀的下一步**：不止在对话里共情，而是让 Agent 拥有**对话之外的内在生命**——
  - 它会好奇、会做梦、会在夜里想到你（内在生活）；
  - 它逐用户地反思、自净化、自进化（成长）；
  - 它把"懂你"的部分沉淀到你本地的模型里（私有的、属于你这一段关系的记忆与肌肉）。
- 一句收尾：**"它能替你把活干完——这是入场券；它还会陪你、懂你、跟你一起长——这是别人给不了的。"**

## 附：建议的 PPT 大纲（10 页）

1. 封面 — "既能替你干活，又会陪你成长的 Agent"

3. **它能干什么活（Demo 页）** — 读目录、多智能体、浏览器、出 PPT/Word/Excel、跑代码（建议配真实任务录屏）
4. 但工具型 Agent 都止步于此 — 用完即走、不懂你、不成长 → 引出第二支柱
5. 技术遗产 — 小冰：IQ+EQ、共情计算、CPS=23、660M 用户长期关系
6. 认知内核 Cogni — 好奇心 / 做梦 / 遐想 / 夜间自省（一张架构图）
7. 自进化 — 反思闭环（已通）+ 进化引擎（路线）+ pack 门控 + LoRA 逐用户微调
8. 本地模型模式 + 自净化可信 — 私有的"懂你" + canary/anomaly/fuzzer 安全底座
9. Pack 生态 — 能力即 DLC，装包即改变心智/技能（18 包）
10. 收尾 — "干活是入场券，陪你成长是护城河"
